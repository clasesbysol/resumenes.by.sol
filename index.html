Comentarios públicos: cómo lo podemos hacer

Sí se puede, pero GitHub Pages solo muestra páginas: no guarda comentarios por sí mismo.

Opciones buenas:

  • Giscus: queda público usando GitHub Discussions. No es anónimo: la persona comenta con GitHub.
  • Formulario anónimo: se puede mandar a Google Forms o similar, pero para que aparezca publicado hay que conectarlo o revisarlo.
  • Disqus/Cusdis: sistema externo de comentarios. Puede permitir nombres sueltos, pero depende del servicio.

Mi recomendación: para empezar, usar un formulario anónimo para recomendaciones y después vemos si conviene mostrarlo automático o moderado.
